How CloudInfra Deploy Engine works

Step 1
Pick a mode

Choose between a pre-built template or Custom ZIP mode. Templates hide Terraform complexity behind simple, focused forms.

Step 2
Provide inputs & AWS keys

Fill in instance names, CIDRs, GitHub repo URLs or DB details. Add AWS keys that are used only at execution time.

Step 3
Run & observe

Terraform runs in a controlled environment. Watch live logs, inspect outputs, and tear everything down with a single Destroy action.
